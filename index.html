How Your Shipment Gets Booked

A simple guide to how pricing, booking, and payment work together on the platform.

Step-by-Step Journey
📦

Step 1 — Enter Shipment Details

The user fills in basic information about the package they want to ship — where it's going, how heavy it is, and what's inside.

Destination Country Package Weight (kg) Dimensions (L × W × H) Document or Parcel?
💰

Step 2 — System Calculates Shipping Price

AeroTrek automatically computes the best price from multiple courier companies and shows all available options sorted from cheapest to most expensive.

How the price is calculated:
If a box is 30×20×10 cm, the system calculates its "size weight" as 30×20×10 ÷ 5000 = 1.2 kg. It then compares this with the actual weight and uses whichever is higher — this is the billable weight.
DHL FedEx Aramex UPS + 6 more carriers
👆

Step 3 — User Selects a Carrier & Confirms

The user reviews the list of available carriers with their prices and delivery times, then picks one and clicks "Book".

Nothing is charged yet. Viewing prices is always free. The wallet is only touched when the user actually confirms the booking.
🔒

Step 4 — System Runs 3 Safety Checks

Before anything is booked or charged, the system automatically verifies three things:

Check 1 — Is the user logged in?
The system confirms the user's session is active and valid.
Check 2 — Is KYC verified?
International shipping requires identity verification (Aadhaar / PAN / GST). If not done, booking is blocked.
Check 3 — Does the wallet have enough balance?
If the shipping price is ₹2,000 but the wallet has ₹800, the booking is rejected immediately with a message to recharge.
🚀

Step 5 — Shipment is Created with Courier

All three checks pass. AeroTrek sends the shipment details to the courier company's system (DHL, FedEx, etc.) and gets back a tracking number and shipping label.

AWB Number (tracking) Shipping Label (PDF) AeroTrek Booking ID
✅

Step 6 — Wallet is Deducted

Only after the courier confirms the booking successfully, the exact shipping price is deducted from the user's AeroTrek wallet. The transaction is saved and linked to the shipment permanently.

Important: If the courier fails to create the shipment (e.g. a network error), the wallet is NOT deducted. Money is only taken when the booking is 100% confirmed.

Note — This Flow May Change After Shiprocket Integration

Currently AeroTrek connects directly to couriers (DHL, FedEx, Aramex, UPS, etc.) through the Overseas API. Once Shiprocket is integrated, some parts of this flow will be updated:

  • Price calculation may pull rates from Shiprocket instead of AeroTrek's own rate tables
  • Shipment creation will go through Shiprocket's system, not directly to the courier
  • Tracking numbers and labels will be generated by Shiprocket
  • The wallet deduction logic and balance check will remain the same
  • An updated diagram will be provided once Shiprocket is live
